# Macro Data System

An automated, data-driven analytical platform that programmatically fetches, parses, and manages raw sovereign intelligence across 9 major economic powers. 

## Project Specifications & Details
* **Automation Lifecycle:** The core tracking loop checks and builds clean internal static JSON storage configurations automatically every **3 months**.
* **Zero Simulation Rule:** The system completely drops simulated calculations in favor of raw data fields mapped directly from open-source web standard APIs.
* **Architecture Style:** Built with an decoupled Single Page Application view interface (`index.html`) running alongside a Node.js data worker pipeline.

## System Features & Key Advantages
1. **Isolated Performance Storage:** By assigning databases into 9 standalone files (`usa.json`, `deu.json`, etc.), the application eliminates single point of database failures and query slowdowns.
2. **Instant Local Exports:** Users can click to save a standalone compilation file directly via the browser without requiring round-trip cloud render steps.
3. **Multi-Page Semantic Print Formatting:** The styling layer formats your screens into 6 clean printable sheets via standard operating system print dialogues (`Ctrl+P` / `Cmd+P`).

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the background database synchronization worker:
   ```bash
   npm start
   ```
3. Open `index.html` directly in any web browser to view your clean dashboard presentation screens.
